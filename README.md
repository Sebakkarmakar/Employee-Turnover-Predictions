📊 Employee Turnover Prediction App 👨‍💼
🚀 Project Overview

This project predicts whether an employee is likely to leave or stay in an organization based on various factors such as job satisfaction, salary, work-life balance, and more.

The project covers the complete ML lifecycle:

Data preprocessing
Model building
Hyperparameter tuning
Deployment using a web app
🎯 Objective

To help organizations identify employees at risk of leaving and take proactive actions to improve retention.

🧠 Machine Learning Models Used
Logistic Regression (L1 & L2 Regularization)
Random Forest
⚙️ Techniques Applied
Data Cleaning & Preprocessing
Feature Engineering
Feature Scaling using MinMaxScaler
Hyperparameter Tuning with GridSearchCV
📈 Model Performance
✅ Best Model: Random Forest
🎯 Accuracy: ~91%
📊 Evaluated using Confusion Matrix & Accuracy Score
💻 Web App (Streamlit)

An interactive app built using Streamlit that allows users to:

✔ Enter employee details
✔ Get real-time predictions
✔ View whether an employee is likely to leave

🛠 Tech Stack
Python 🐍
NumPy, Pandas
Scikit-learn
Streamlit
